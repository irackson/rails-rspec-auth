# RAILS RSPEC AUTH _by Ian Rackson_

## A 2 page (sign in, dashboard) ROR application made using devise

### with rspec testing on controllers and views

#### To run locally:

    1. `yarn install`
    2. `bundle install`
    3. `rails db:create`
    4. `rails db:migrate`
    5. `rails s`

#### To run tests: `rails spec`
